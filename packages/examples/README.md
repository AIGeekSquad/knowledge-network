# Knowledge Network Examples

**Interactive examples demonstrating library features**

This package contains interactive examples showcasing the capabilities of the `@aigeeksquad/knowledge-network` library.

---

## Running Examples

### Development Server

```bash
# From project root
pnpm dev

# Or from this directory
cd packages/examples
pnpm dev
```

Open http://localhost:5173 to view the examples.

### Available Examples

- **Basic Graph** - Simple network with interactive nodes
- **Edge Bundling Showcase** - Comparison of simple vs bundled edges
- **Large Network Performance** - 1000+ nodes with optimized settings
- **Custom Styling** - Data-driven styling and theming
- **Dynamic Updates** - Real-time graph modifications
- **Framework Integration** - React, Vue, and Angular integration examples
